## 💡 实用技巧

### 1. 创建别名（简化使用）

在 `~/.zshrc` 或 `~/.bash_profile` 添加：

```bash
alias emuera='/Users/ss/Documents/Project/iOS/emuera-mac/启动Emuera.sh'
```

然后：
```bash
source ~/.zshrc
emuera demo
```

### 2. 拖拽运行

在Finder中：
1. 将脚本文件拖到终端窗口
2. 自动获取完整路径
3. 运行 `./启动Emuera.sh run [拖入的路径]`

### 3. 快速测试

创建测试脚本：
```bash
echo 'PRINTL 测试成功！QUIT' > test.erb
./启动Emuera.sh run test.erb
```
